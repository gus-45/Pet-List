# Pet-List

### 📱 Aplicativo de Gerenciamento de Pets

Projeto desenvolvido como parte da disciplina de **Dispositivos Móveis** utilizando React Native e Expo.

## 📋 Sobre o Projeto

Este aplicativo foi desenvolvido através de três exercícios práticos que implementam funcionalidades essenciais de uma aplicação mobile.

## 🎯 Exercícios Implementados

### Exercício 1 - Cadastro de Pets
- ✅ Lista de pets cadastrados
- ✅ Formulário com campos de nome e descrição
- ✅ Botão para adicionar novo pet
- ✅ Validação: nome deve ter no mínimo 2 letras
- ✅ Validação: não permite nomes duplicados

### Exercício 2 - Exclusão e Estilização
- ✅ Botão de exclusão individual para cada pet
- ✅ Estilização completa da interface
- ✅ Design responsivo e intuitivo

### Exercício 3 - Navegação entre Telas
- ✅ Tela de listagem de pets
- ✅ Tela de adicionar pet
- ✅ Navegação entre as telas com botões
- ✅ Fluxo de navegação bidirecional

## 📦 Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/gus-45/Pet-List.git
```

2. Instale as dependências:
```bash
cd nome-do-projeto
npm install
```

3. Execute o projeto:
```bash
npx expo start
```

4. Escolha onde executar:
   - Pressione `a` para Android
   - Pressione `i` para iOS
   - Pressione `w` para Web
   - Ou escaneie o QR Code com o app Expo Go

## 📱 Funcionalidades

- Cadastro de pets com nome e descrição
- Validação de dados em tempo real
- Listagem de todos os pets cadastrados
- Exclusão de pets da lista
- Navegação fluida entre telas
- Interface intuitiva e responsiva
